Game of life

Conways Game of life is fascinating to me, how such easy/simple rules can allow such complex structures to evolve. In this project I focused on learning some python methods for:

  • Testing
  • Profiling
  • Optimizations

    • Since Game of life has a lot of mathematical evaluations to calculate I'd reckon that I could optimize the program by compiling python code to C code. Fortunately there was a module called Cython which did just that. Unfortunately half of time went into making cython work with my windows OS, but it work in the end and I'm happy with the result. The speed of the program increased by ~2.4x times just by compiling to C . I was also introduced to markdown code and github commands.
